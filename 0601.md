 ## 67. Add Binary
 - Bit-by-bit Computation
 ```py
 class Solution:
    def addBinary(self, a: str, b: str) -> str:
        res = []
        carry = 0
        
        i = len(a) - 1
        j = len(b) - 1

        while i >= 0 or j >= 0 or carry:
            bit_a = int(a[i]) if i >= 0 else 0
            bit_b = int(b[j]) if j >= 0 else 0
            bit_sum = bit_a + bit_b + carry

            cur_bit = bit_sum % 2
            carry = bit_sum // 2

            res.append(str(cur_bit))

            i -= 1
            j -= 1

        return "".join(res[::-1])

        # time: O(n)
        # space: O(1)
 ```
 
- Bit: bitwise
 ```py
 class Solution:
    def addBinary(self, a: str, b: str) -> str:
        x = int(a, 2)
        y = int(b, 2)
        
        return bin(x + y)[2:]
 ```
