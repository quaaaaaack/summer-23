## 13. Roman to Integer
```py
class Solution:
    def romanToInt(self, s: str) -> int:
        roman_dict = {
            'I': 1,
            'V': 5,
            'X': 10,
            'L': 50,
            'C': 100,
            'D': 500,
            'M': 1000
        }

        res = 0
        pre = 0
        for ch in s[::-1]:
            cur = roman_dict[ch]
            if pre > cur:
                res -= cur
            else:
                res += cur
                pre = cur
        
        return res

        # time: O(n)
        # space: O(1)
```
